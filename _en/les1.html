First of all, let's learn to greet!

{% include audio.html file="1-001" %} Мшыбзи'а! Good day! Hello! Literally: the day is good!
{% include audio.html file="1-002" %} Бзи'а(ра) уба'аит! Answer greeting (to a man). Literally: yes to see you (a man) is good!
{% include audio.html file="1-003" %} Бзи'а(ра) бба'аит! Answer greeting (to a woman). Literally: yes to see you (a woman) is good!

(The brackets show an optional part, i.e. in any situation we can read бзиара убааит the same as бзиа убааит).

And now we will introduce the Abkhazian alphabet.

Abkhazian has 6 vowels and 56 consonants. In total 62. Of these 26 are from the Russian alphabet, i.e. all letters except ё, й, щ, ю, я, э, ъ are included in the Abkhazian alphabet.

Let's learn a few common phrases.

{% include audio.html file="1-201" %} Сара' сцо'ит - I'm going, I'm leaving
{% include audio.html file="1-202" %} Сара' сгы'лоуп - I'm standing up
{% include audio.html file="1-203" %} Сара' избо'ит - I see
{% include audio.html file="1-204" %} Сара' бзи'а избо'ит са'ни са'би - I love my mother and father
{% include audio.html file="1-205" %} Сар'а иаа'схәоит ача' - I am buying bread

At the end of the alphabet are two letters - Ь (soft sign) and Ә (rounding sign). These symbols don't have their own sound, but in combination with other letters form new sounds.
Compare:

{% include audio.html file="1-301" %} к - кь - кә
{% include audio.html file="1-302" %} г - гь - гә
{% include audio.html file="1-303" %} х - хь - хә

Soft consonants in Abkhazian (except ч, ҷ) are accompanied by a soft sign.

The sounds кь, гь, хь are pronounced approximately  the same as the Russian consonants к, г, х + vowel sound и. For example: Abkhazian гь - Russian combination ги (гиря); Abkhazian кь - Russian ки (кино); Abkhazian хь - Russian хи (хитрый).

Adding the soft sign ь to the letter к gives the letter {% include audio.html file="1-401" %} кь.
For example:
{% include audio.html file="1-402" %} а'нкьа - before (earlier) (a long time ago)
{% include audio.html file="1-403" %} макьа'на - still (for the time being)

Adding the rounding sign ә to the letter к gives the letter {% include audio.html file="1-501" %} кә.
Unlike the Russian combination ку, this sound is pronounced together кә.
{% include audio.html file="1-502" %} акәа'ц - meat
{% include audio.html file="1-503" %} а'кәашара - to dance

And so: {% include audio.html file="1-301" %} к - кь - кә

And similarly for the other letters. For example:

{% include audio.html file="1-302" %} г - гь - гә
{% include audio.html file="1-602" %} агара' - to take, to take away
{% include audio.html file="1-603" %} агьа'ма - taste
{% include audio.html file="1-604" %} агәы' - heart

{% include audio.html file="1-303" %} х - хь - хә
{% include audio.html file="1-702" %} ахы' - head
{% include audio.html file="1-703" %} иахьа' - today
{% include audio.html file="1-704" %} ахь - to (suffix)
{% include audio.html file="1-705" %} ахә - price

(If the suffix ахь follows a word that ends in a vowel (а,ы), then it is written together with the word it follows. Compare: ашкол ахь, but аусурахь "(to go) to work" (аусура + ахь)).
