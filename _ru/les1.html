В самом конце алфавита стоят две буквы - Ь (мягкий знак) и Ә (знак огубления). Эти знаки не имеют своего звука, но в сочетании с другими буквами образуют новые звуки.
Сравните:

{% include audio.html file="1-301" %} к - кь - кә
{% include audio.html file="1-302" %} г - гь - гә
{% include audio.html file="1-303" %} х - хь - хә

Мягкие согласные в абхазском (за исключением ч, ҷ) сопровождаются мягким знаком.

Звуки кь, гь, хь произносятся приблизительно  так же, как и русские согласные к, г, х + гласный звук и. Например: абхазский гь - русское сочетание ги (гиря); абхазский кь - русское ки (кино, кисть); абхазский хь - русское хи (хитрый, хилый).

Прибавляем к букве к мягкий знак ь и получаем букву {% include audio.html file="1-401" %} кь.
Примеры:
{% include audio.html file="1-402" %} а'нкьа - раньше; когда-то
{% include audio.html file="1-403" %} макьа'на - пока еще

Прибавляем к букве к знак огубления (ә) и получаем букву {% include audio.html file="1-501" %} кә.
В отличие от русского сочетания ку, данный звук произносится слитно кә.
{% include audio.html file="1-502" %} акәа'ц - мясо
{% include audio.html file="1-503" %} а'кәашара - танцевать

Итак: {% include audio.html file="1-301" %} к - кь - кә

То же происходит и с другими буквами. Например:

{% include audio.html file="1-603" %} агьа'ма - вкус
{% include audio.html file="1-604" %} агәы' - сердце

{% include audio.html file="1-702" %} ахы' - голова
{% include audio.html file="1-703" %} иахьа' - сегодня
